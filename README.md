"CODEX MORTIS - Unus Vita, Unus Mors, Decem Linguae"
# CODEX MORTIS

> *"Unam vitam habeo, unam mortem.  
> Hoc Codex Mortis erit testamentum meum.  
> Linguas decem in unum congregabo,  
> Et chaos ordinem pariet."*

**Перевод:**  
*"Одну жизнь имею, одну смерть.  
Сей Кодекс Мёртвых будет завещанием моим.  
Языки десять воедино соберу,  
И хаос породит порядок."*
